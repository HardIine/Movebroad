# 一、巡检车
1. 产品功能及产品原理 \
   功能：帮助车位管理员记录车辆停泊位置以及时间和车牌信息 \
   原理：

    - 利用车位检测感知状态
    - 三方位相机抓拍识别车牌
    - 定位系统定位车辆停车位信息
    - 终端移动网络上传数据至管理平台
1. 产品关键参数 
   - 车辆参数：  \
    品牌:雅迪、爱玛、绿源 \
    最高续航及速度及标配电池：80KM 、 25KM/h 、48V24AH 
    - 定位单元： \
    信号：四星八频或者五星十六频 \
    首次定位时间及重捕率：40s 、 <1s
    - 性能参数： \
    供电电源及功耗：12V30AH、 <30W \
    工作时长：大于24小时 \
    车辆捕获率：98% \
    泊位匹配正确率：99% \
    车牌识别准确率：99% \
    综合：96% \
    传输协议：TCP、HTTP \
    工作环境：-20℃-75℃ \
    充电环境：0℃-60℃ \
    IP防护等级：IP67
2. 定位产品连接
   1. 打开泊位app并登陆
   2. 标定设备的安装，先将 GNSS 天线与定位杆用螺丝旋钮相连，再将 GPS 主控器的 GNSS 接口与 GNSS 天线连接，COM2 接口与标定泊位工具连接，最后将电源与 GPS 主控器 COM1 相连接，完成设备的安装。
3. 泊位标定
   1. 打开串口
   2. 设置增量参数 \
      填写真实泊位编号以及虚拟编号，并且设置递增或者递减
   3. 选择路段
   4. 设置泊位起点 \
      调节定位杆黑色旋钮，升高 GNSS 天线至合适高度；将定位杆移动至泊位外侧端点，保持与地面垂直一段时间，同时观察标定泊位工具左上角信息栏中“当前坐标”与“定位质量”，确认无误后，点击设置为泊位起点
   5. 设置泊位终点 \
      再将定位杆移动至泊位外侧另一个端点上，保持与地面垂直一段时间，同时观察标定泊位工具左上角信息栏中“当前坐标”与“定位质量”，确认无误后，点击设置为泊位终点
   6. 添加泊位：观察泊位长度与实际长度是否相当，一致则添加，不一致则为异常数据，点击重置
   7. 输入泊位信息：根据现场情况输入泊位编号、虚拟泊位编号、选择是否为首尾泊位，确认数据无误后，点击“添加泊位”
4. 泊位注意事项
   1. 如果两泊位之间存在间隔，需根据现场情况必须设置首尾泊位；
   2. 标定泊位过程中，应当手动记录泊位信息，尤其是首尾泊位信息；
   3. 获取泊位地理位置时，定位杆需要在地面停留 3-4 秒后，方可添加泊位；
   4. 添加泊位时，定位杆应与地面垂直并保持不动直至此泊位编号添加完成；
   5. 泊位的地理位置的定位质量需要为 4；
   6. 标定泊位时，需要注意两泊位编号之间的泊位长度，泊位长度应与实际长度相符，当出现异常长度时，需重置此泊位的泊位编号，重新标定泊位。
   7. 确定泊位起点与终点时，若该点被障碍物遮挡，可选择升高定位杆，或者选择最靠近泊位一点（一般以不减少泊位长度为前提）；若距离实在偏差过大，可视情况能否让车主挪车;
   8. 首尾泊位在标定过程中，沿着路的方向延长 0.5 米，使泊位能够更好的匹配到
5. 巡检方法及注意事项
   1. 粗调相机：前相机根据图像来判断相机角度；中相机向下卡死，摆
正相机；后相机在后方、下方卡死
   2. 设置以太网IP：以太网2的IPv4ip地址设置为192.168.1.2，子网掩码为255
   3. 打开视频流预览工具打开相机
   4. 精调相机：
      前轮在泊位的后标志线的延长线上，巡检车距离泊位一个巡检距离（1.5m）。由一人坐在巡检车上，呈巡检姿势，双脚落地支撑 \
      前相机：拧松螺丝，使相机在车牌在第三象限靠中间并且车牌摆正，清楚看到泊位编号。 \
      中相机：拧松螺丝，使相机在车牌在正中区域并且车牌摆正，清楚看到泊位编号。 \
      后相机：拧松螺丝，使相机在车牌在第四象限靠中间并且车牌摆正，清楚看到泊位编号。
   5. 移动视频巡检设备开机 \
      打开巡检车尾箱，按下主机开关按钮，扬声器会播报“欢迎使用目博科技巡检终端，系统初始化成功，GPS 初始化成功，前摄像头已连接，中摄像头已连接，后摄像头已连接”。
1. 巡检注意事项 \
   （１）当前方车位正在有车停入时，可缓慢停下等车辆停放好后继续巡检； \
   （２）当前方车位有车并且侧边也临时停放车辆，此时观察两车间距，如适合巡检车通行，建议从两车中间通行，若间距不足以通行，则建议绕到临时停放车辆前通行，同时注意此时巡检记录，如有异常则建议关注此时的巡检记录； \
   （３）巡检开始时，巡检车与泊位平行并从距离首泊位 2m 以上的位置触发； \
   （４）巡检过程中，巡检车与泊位始终保持 1.5-2m 的距离，保证巡检车可以正常拍摄到车牌、车身和泊位编号； \
   （５）巡检结束时，巡检车与泊位平行并在距离尾泊位 2m 以上的位置后再进行转向或者结束巡检任务； \
   （6）如果免费时间为30分钟，则建议每15-20分钟巡检一次。

# 二、车位锁
1. 主要实现功能  \
   通过微波雷达+地磁检测车辆，对车位占用状况进行实时监控和智能管理，结合扫码开锁实现APP预约+自动落锁+自动计时+车走自动上锁收费等功能，实现车位查找、 预约、导航、支付、共享和智能控制管理。
2. 工作原理 \
   通过微波雷达+地磁检测车辆或者遥控信号，对车位占用状况进行实时监控和智能管理
3. 产品重要参数 \
   电源及容量：可充电蓄电池， 10Ah \
   工作环境：-40℃-80℃ \
   待机电流：8mA \
   通讯方式：4G/RF-433MHz \
   遥控距离：12m \
   抗压性能：大于5t \
   防护等级：IP67 \
4. 测试方法 \
   具体实际测试，使用遥控器使车位锁抬起或者降落，在车上装感应器，或者小程序控制
5. 安装方法 \
   一般安装在车位入口三分之一处
   1. 使用记号笔对安装位置标记
   2. 对之前标记处打孔
   3. 放入膨胀螺丝拧开螺帽
   4. 对准车位锁上的孔拧紧螺丝
6. 常见问题及回答 \
   车位锁地磁产品QA(1).doc
# 三、地磁产品
1. 地磁编号定义
2. 地磁原理：通过地磁+微波双重检测方式，判定停车位上车辆变化情况
3. 产品及相关配套工具
   1. 安装小程序：用于安装时输入信息绑定设备和泊位
   2. 售后管理平台：管理安装时录入的信息
   3. 智慧停车管理平台：管理停车场和停车位信息
   4. 目博云停车：通过小程序查看设备状态及信息
   5. 蓝牙配置器：核实编号以及调试设备
4. 重要参数
   1. 电源：内置3.6V锂电池
   2. 电池容量：≥38Ah
   3. 待机电流：≤15μA
   4. 功耗：60uW
   5. 产品寿命：大于等于三年
   6. 探测半径：0.5m
   7. 工作温度：-40-85℃
   8. 防护等级：IP68
   9. 抗压性能：大于等于10t
   10. 检测方式：产品通过地磁+微波双重检测方式，判定停车位上车辆变化情况
   11. 检测准确率：大于等于99%
   12. 心跳间隔：1-240min可调
   13. 标准质保：24个月
5. 安装
   1. 安装位置 \
      平行泊位安装在靠近道路外侧三分之一位置处，垂直泊位安装在泊位中间
   2. 安装方法 \
      1、使用直径 12cm 的水钻，地面打孔：(直径)12cm * (深度)11cm 。 \
      2、打孔后：底部垫细沙，注入速干水泥。 \
      3、安装前：安装前翻转 6 次确保激活设备，听到滴滴声音，即确认地磁激活。 \
      4、安装：(确保激活后，未激活不得放入)将地磁放入承托中，然后将地磁和承托整体放入孔中，按压地磁，确保承托紧贴地面。 \
      5、事后处理：揭开地磁上表面粘贴的塑料保护膜，确保地磁上表面干净。 \
      6、扫描安装小程序二维码，进入微信小程序，登录安装人员管理账号，扫描地磁标签的二维码，输入泊位号，拍地磁条形码和泊位号照片各一张，确认提交完成即可。
   3. 拆除方法 \
      1、使用镊子，将地磁上表面的三个圆孔内的堵头取出。 \
      2、使用特制扳手，将扳手的三颗铆钉与地磁的三个圆孔一一对应。 \
      3、逆时针旋转扳手，将地磁内胆取出即可。 \
      4、取出后可以倒置存放。如果后续安装使用，可以重新采购地磁外壳。
      5、拆除之后需要返场检修，检测密封性、功能性以及补贴设备标签。
   4. 设备安装 \
      进入目博售后助手，登录提供的账号，点击设备安装。先选择车场（也可以选择新建），再输入泊位号，再扫描设备号，还需要上传两张照片，一张为泊位号，一张为设备照片，点击提交。
   5. 售后管理平台 \
      登录账号后可以创建停车场，还可以再平台上添加或者修改设备信息，可以查询已经安装的设备的信息，还可以导出信息。  
   6. 智慧停车管理平台 \
      可以新建用户，查看或者新建修改停车场信息，查看或者新建修改停车位信息。还可以批量导入信息，选择需要上传的信息的对应模板下载，按照模板中的要求填写后可以批量导入信息，导入之后可以查看任务创建结果，在错误信息一栏中可以看到错误的信息以及错误的原因。
